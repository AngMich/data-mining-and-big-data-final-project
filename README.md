This project presents a hybrid machine learning framework for network intrusion detection on the NSL-KDD benchmark dataset. Unsupervised K-Means clustering is applied to derive structural latent features — cluster assignments and Euclidean distances to centroids — which augment a supervised Random Forest classifier trained on the top 20 mutual information-selected features. The model achieves 98.78% validation accuracy and 80.13% test accuracy on KDDTest+, with precision of 97.06% and F1-score of 79.36%.
